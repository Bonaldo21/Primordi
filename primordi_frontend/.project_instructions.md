# Primordi - Loja de Couro Artesanal

## Overview
Frontend e-commerce para a marca Primordi (produtos de couro artesanais). Design minimalista inspirado em Ocean & Oak.

## Technical Approach
- Pure frontend app consuming Java/Spring backend at `http://localhost:8080/api`
- No internal database - all data from backend API
- Auth via JWT stored in localStorage (backend handles auth)
- Cart managed in localStorage (no backend cart endpoints)
- Sample data fallback when backend unavailable

## API Notes
- Backend has doubled `/api` prefix for some endpoints: enderecos, pedidos, fretes use `/api/api/...`
- CORS configured for localhost:3000
- JWT Bearer token auth

## Visual Style
- **Fonts**: Cormorant Garamond (display/headings), DM Sans (body)
- **Colors**: Warm earth tones - cream background (#FAF7F2), cognac primary (#8B6F47), dark brown foreground
- **Design**: Minimal, clean, lots of whitespace, subtle shadows, warm neutral palette
- Light/dark mode supported

## Page Structure
- `(store)` route group: Home, Catálogo, Produto/[slug], Carrinho, Checkout, Sobre
- `(auth)` route group: Login, Registro
- `/admin` section: Dashboard, Produtos, Categorias, Pedidos, Clientes
- Admin protected by role check in layout

## Product Images
- 12 AI-generated product images hosted on CDN (bolsas, carteiras, cintos, etc.)
- Hero image and About section image also generated

## Key Patterns
- AuthContext for JWT management
- CartContext for cart state (localStorage)
- API service in lib/api.ts with typed methods
- Sample data fallback in lib/sample-data.ts
- Format utilities in lib/format.ts
